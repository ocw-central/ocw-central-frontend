# OCW-CENTRAL-FRONTEND

## Description

Hi, this is a repo for the frontend implementation on ocw-central.

## How to develop

- Install dependencies:

  ```bash
  yarn install
  ```

- Start the server

  ```bash
  yarn dev
  ```

- Use eslint to lint the code

  ```bash
  yarn lint
  ```

- Check for build errors before opening PR

  ```bash
  yarn build
  ```
